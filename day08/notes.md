### Basic If Statement
```python
if condition:
    # code
```

### If-Else
```python
if condition:
    # code
else:
    # code
```

### If-Elif-Else
```python
if condition1:
    # code
elif condition2:
    # code
else:
    # code
```

### Nested Conditionals
```python
if condition1:
    if condition2:
        # code
```

### Ternary Operator
```python
value = a if condition else b
```

### Multiple Conditions
```python
if age >= 18 and has_license:
    # code

if x < 0 or x > 100:
    # code
```
